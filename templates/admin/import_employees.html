CSV Format Instructions

The CSV file must contain the following columns (in any order):

  • name – Full name of employee
  • email – Unique email address
  • password – Plain text password (will be hashed)
  • position – Job position (optional, default: Staff)
  • daily_rate – Daily salary in PHP (optional, default: 500)
Example CSV content:
name,email,password,position,daily_rate
John Dela Cruz,john.dela@example.com,pass123,Sales and Marketing Staff,550
Maria Santos,maria.santos@example.com,pass456,HR Officer,600
Carlos Reyes,carlos.reyes@example.com,pass789,Delivery Driver,480
Note: Duplicate emails will be skipped automatically. The system will report how many employees were successfully imported.
